Shabad OS

Overview

This program replicates many functions of SikhiToTheMax. It is many times faster at searching, displaying, and navigating most shabads. New features, unavailable in other apps, include: Showing the next line on the display; Automatically selecting the repeated lines and the rest of the shabad; and, Controlling or duplicating the program on smartphones, tablets, and laptops over the same network/wifi.

Features

Search defaults to first letter search. Using a "#" symbol at the start will switch to full word search.

  • See results as you type them
  • Use blank space characters as wildcards (helps with similar sounds like vismaad and bismaad)
  • Get short results earlier (making those two or three word lines much easier to find)

Shabads:

  • Each line has it's own key assigned. The first ten are 1, 2, 3, ..., 0. Afterwards, we use Q, W, E, ..., P for the next ten. You'll see that each letter is shifted over a bit from the numbers above them, just add ten to each. Compiled Banis have special use of this, for instance in Asa di Var it will instead go to the related stanza. The 24th (last) stanza of Asa di Var can be accessed with the "F" key.
  • You can tap the spacebar key or click "Set Main" at the bottom to set the line you're currently on as the main line. Tapping the spacebar or clicking "Toggle Line" will automatically move to the first line of the shabad. Now if you want to return to the main line, simply toggle again. If you do it a fourth time, now from the main line, it will smartly return to the next line you want to read.

Tutorial

On first run, you'll be greated with a blank window (display) with a smaller embedded window (navigator) in the bottom right. You can hover over any button for a description in it's titlebar. For instance hover over the second to last icon in the navigator's titlebar. This button will let you minimize the navigator.

There are many shortcuts available. If you ever want to look up a shortcut, type the question mark (?) key on your keyboard (shift + /). This will bring up a menu showing you the various shortcuts.

To connect with a mobile device, hover over the titlebar of the display window. This will get you the web address to type into your phone. You can similarly join this address on different laptops, tablets, or even smart TVs. A couple noteworthy addresses include adding "/display" or "/kobo" to the end of the program's address. Example: 192.168.X.XXX:8080/display. You can use this to clone the display. The kobo link can be used on modern E-Ink technologies, similar to the Kindle, as an alternative view to reading the shabad.
